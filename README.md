# 📚 Research Paper Analyzer + Chapter Summarizer + Q&A Chat

A multi-mode AI agent that lets you analyze, summarize, and chat with any PDF book or research paper — powered by **Groq (LLaMA 3.3 70B)** and **LangGraph**.

---

## 🚀 Features

| Mode | Description |
|------|-------------|
| **Full Analysis** | Multi-agent pipeline: extracts key findings, writes executive summary, pulls citations, and generates insights |
| **Chapter Summary** | Pick any chapter and get an instant structured brief (core message, key ideas, takeaway, best quote) |
| **Q&A Chat** | Conversational chat interface — ask anything about the book/paper |

---

```env
GROQ_API_KEY=your_key_here
```

3. Place your PDF in the same directory as `agent.py`

## 🤖 How It Works

### Mode 1 — Full Analysis (LangGraph Multi-Agent Pipeline)

```
Boss → Analyzer → Review → Summary → Review → Citations → Review → Insights → Review → Combine
```

Each agent is reviewed by a scoring agent (1–10). If the score is below 7, the agent retries (max 2 retries). The final output is saved to `research_brief.txt`.

### Mode 2 — Chapter Summary

Extracts the chapter text by page range, sends it to Groq, and returns a structured brief:
- **Core Message** — single most important idea
- **Key Ideas** — 5 bullet points
- **Practical Takeaway** — real-life application
- **Best Quote or Example**

Output is saved to `chapter_N_summary.txt`.

### Mode 3 — Q&A Chat

Maintains a conversation history (last 20 messages) with the full paper loaded as context. Supports special commands:

| Command | Action |
|---------|--------|
| `summarize` | Get a 150–200 word summary of the whole book |
| `clear` | Clear conversation history |
| `quit` / `exit` / `q` | Exit chat mode |

## 🧠 Model

All API calls use:
- **Model:** `llama-3.3-70b-versatile`
- **Provider:** Groq
- **Max tokens:** 2000 (analysis), 1500 (chat)
- **Temperature:** 0.3 (analysis), 0.5 (chat)

## 🔧 Customization

### Use a different PDF
Just drop any PDF into the script directory and run `agent.py`. For multiple PDFs, you'll be prompted to pick one. To hard-code a specific file:

```python
pdf_path = find_pdf("my_paper.pdf")
```

### Change the chapter map
Edit `CHAPTER_MAP` at the top of `agent.py` to match your book's structure:

```python
CHAPTER_MAP = {
    1: {"title": "Introduction", "pages": (1, 10)},
    2: {"title": "Methods",      "pages": (10, 20)},
    ...
}
```

### Use a different model
Change the `MODEL` constant:

```python
MODEL = "llama-3.3-70b-versatile"  # or any Groq-supported model
```
